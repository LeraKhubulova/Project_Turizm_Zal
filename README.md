Тема: Туризм Название: Интерактивный музей

Цель: разработать сайт, который позволяет пользователю онлайн рассматривать залы и экспонаты, узнавать их историю и т.д

Рсновные задачи:

1)Реализовать главную страницу с тремя (пока неточно) залами.при нажатии на которые пользователя будет перебрасывать на отдельную страницу зала
2)Реализовать страницу зала, с более подробной информацией о нём и самих экспонатах. На странице будет присутсвовать фотография зала, при наведении курсора мыши на экспонаты они будет увеличиваться в размерах и подсвечиваться. При нажатии пользователя будет перебрасывать на страницу с подробной информацией об конкретном экспонаты
3)Реализовать страницы экспонатов
4)Реализовать регистрацию через JSON. Может иметь два состояния работник музея (доступ к редактирвоанию) и просто посетитель
